## Example: Welly on Film

**Product Name**: Welly on Film
**One-liner**: A community platform for analog film photography competitions in Wellington
**Target User**: Local film photography enthusiasts in Wellington, NZ

### Core Purpose

Wellington has a vibrant analog photography community but no dedicated space to share work and connect. Welly on Film creates a micro-social platform where locals can submit their best film shots monthly for potential publication in a community magazine.

### Key Features

1. **Monthly Submissions**: Upload up to 3 analog photos per month
2. **Community Gallery**: Browse photos shared by other Wellington photographers
3. **Comments & Discussion**: Engage with other photographers' work
4. **Competition Judging**: Community voting and curator selection for magazine features

### Design Principles

- **Photography First**: The images are the hero. Minimal UI that doesn't compete
- **Local Identity**: Feels distinctly Wellington, not generic
- **Analog Warmth**: Digital experience that honors the tactile nature of film

### Technical Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Database**: Supabase
- **Deployment**: Fly.io
- **Storage**: Supabase Storage for images

### Design Language

- **Color Palette**: Warm neutrals, film-inspired accent colors
- **Typography**: Clean sans-serif for UI, subtle serif for editorial moments
- **Tone**: Approachable, community-focused, slightly nostalgic
- **Imagery Style**: Full-bleed photography, generous whitespace

### Success Criteria

1. [ ] Users can sign up and create profiles
2. [ ] Users can upload up to 3 photos per month
3. [ ] Gallery displays all submissions with filtering
4. [ ] Users can comment on photos
5. [ ] Responsive design works on mobile and desktop
6. [ ] Landing page clearly communicates the value proposition

### Out of Scope (v1)

- Direct messaging between users
- Photo editing tools
- Print ordering
- Multiple competition categories
- Monetization features
